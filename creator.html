Guia de Formatação

Tags Básicas

h2 Título Principal

Cria um título grande para seções importantes.

<h2>Título Principal</h2>

h3 Subtítulo

Cria um subtítulo para subseções.

<h3>Subtítulo</h3>

p Parágrafo

Formata um bloco de texto como parágrafo.

<p>Texto do parágrafo.</p>

strong Negrito

Destaca texto em negrito.

<strong>Texto importante</strong>

Elementos Avançados

ul Lista

Cria uma lista não ordenada com marcadores.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

a Link

Adiciona um link clicável.

<a href="https://exemplo.com">Texto do link</a>

img Imagem

Insere uma imagem no conteúdo.

<img src="imagem.jpg" alt="Descrição">

span Texto Colorido

Permite adicionar cor ao texto.

<span style="color: red;">Texto colorido</span>
